# Prowl.Veldrid

A cut up version of ciberman's NeoVeldrid repository - resource binder has been gutted in preference of direct-binding system, where users specify resource indices directly.

This package is intended to be provided as part of Prowl.Graphite, and as such all nuget building has been disabled and gutted. If you want to use Prowl.Veldrid as a standalone package, you should probably look into using NeoVeldrid or a maintained version of an intended standalone Veldrid repository instead. If, for some reason, you prefer manually specifying your resource bindings and attribute locations, you can build Prowl.Veldrid as a package yourself.
